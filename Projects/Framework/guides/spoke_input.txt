Input

Input accepts the arguements varible and prompt,
where the prompt is optional.

The input command takes a user input for the requested prompt,
which defaults to "? " if no prompt is given, 
and assigns it to the given varible.

# let y = 0 
#
# input x # Prints "? " and asks for input
# input y "Y: " # Prints "Y: " and asks for input
